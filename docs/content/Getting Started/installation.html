Installation Guide

Last updated: July 17, 2025 4 min read

Follow these step-by-step instructions to install mnemo on your device. The process is straightforward and should take only a few minutes.

System Requirements

OS: Windows 10 (64-bit) or Windows 11
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: .NET Framework 4.7.2 or later
OS: macOS 10.15 (Catalina) or later
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: Apple Silicon and Intel Macs supported
OS: Ubuntu 18.04+, Debian 10+, or equivalent
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: X11 or Wayland display server

Download Options

Choose the appropriate version for your operating system from our download page.

Installation Steps

  1. Double-click the downloaded mnemo_setup.exe file
  2. If prompted by Windows Defender, click "More info" and "Run anyway"
  3. Follow the installation wizard prompts
  4. Choose your installation directory (default recommended)
  5. Click "Install" and wait for completion
  1. Open the downloaded .dmg file
  2. Drag the mnemo icon to the Applications folder
  3. Right-click mnemo in Applications and select "Open"
  4. Click "Open" when prompted about unverified developer
  5. Grant necessary permissions when prompted

Using .deb (Ubuntu/Debian):

sudo dpkg -i mnemo.deb
sudo apt-get install -f

Using .rpm (Fedora/RHEL):

sudo rpm -i mnemo.rpm

Using .AppImage:

  1. Make the AppImage executable: chmod +x mnemo.AppImage
  2. Double-click to run or execute from terminal

Post-Installation

After installation, you should:

  1. Launch mnemo from your start menu or applications folder
  2. Take the tour to get familiar with the key features

Common Installation Issues

  • Windows Defender blocking: Use "Run anyway" option
  • macOS security warning: Open from Applications folder
  • Linux dependencies: Run sudo apt-get install -f

For more help, visit our troubleshooting guide.
