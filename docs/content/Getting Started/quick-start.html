Quick Start Guide

Last updated: July 17, 2025 3 min read

Welcome to mnemo! This guide will help you get started with the essential features and set up your first study session in just a few minutes.

1. Download and Install

First, download mnemo for your platform from our download page. The installation process is straightforward and should take less than 2 minutes. Please refer to installation guide for more details.

System Requirements: Make sure your system meets the minimum requirements. Most computers from the last 10 years should work perfectly.

2. Take the Tour

We recommend taking a quick tour of the mnemo interface to get familiar with the key features when first starting out.

  • When launching the application for the first time it will ask if you want to take a quick tour. This is recommended to get familiar with the key features, if you are new to the application.
  • If you do skip the tour and regret it later, you can always access it from the settings menu under "About & Support"

3. First Study Session

Mnemo offers several tools to make your study sessions better:

  • Learning Paths - Create learning paths to help you study for exams or other important events.
  • Flashcards - Create flashcards to help you memorize important information.
  • Notes - Create notes to help you organize your study materials.
  • Quizzes - Create quizzes to help you test your knowledge.
  • Mindmaps - Create mindmaps to help you visualize and remember information.
  • Extensions and Games - Mnemo has a built-in system for extensions and games, which you can use to make your study experience a bit more fun.
Extentions and Games: The built-in system for extentions and games are fully open, meaning developers can create their own. If you are interested in creating your own, please refer to the plugins and extensions section.

4. Track Your Progress

Mnemo automatically tracks your study habits and progress, giving you a clear overview of your learning journey. View your stats directly on the dashboard through the statistics widgets.

NOTE: A seperate Analytics page is coming later.

The statistics collected involves:

  • Daily and weekly study time
  • Most productive hours
  • Subject-wise progress
  • Goal achievement rates
Privacy Notice: Mnemo does not collect any personal data, all data is stored locally on your device, except for information parsed to optional AI services.
