Quick Start Guide

Last updated: July 17, 2025 3 min read

Welcome to mnemo! This guide will help you get started with the essential features and set up your first study session in just a few minutes.

1. Download and Install

First, download mnemo for your platform from our download page. The installation process is straightforward and should take less than 2 minutes. Please refer to installation guide for more details.

System Requirements: Make sure your system meets the minimum requirements. Most computers from the last 10 years should work perfectly.

2. Take the Tour

We recommend taking a quick tour of the mnemo interface to get familiar with the key features when first starting out.

  • When launching the application for the first time it will ask if you want to take a quick tour. This is recommended to get familiar with the key features, if you are new to the application.
  • If you do skip the tour and regret it later, you can always access it from the settings menu under "About & Support"

3. First Study Session

Mnemo offers several tools to make your study sessions better:

  • Learning Paths - Create learning paths to help you study for exams or other important events.
  • Flashcards - Create flashcards to help you memorize important information.
  • Notes - Create notes to help you organize your study materials.
  • Quizzes - Create quizzes to help you test your knowledge.
  • Mindmaps - Create mindmaps to help you visualize and remember information.
  • Extensions and Games - Mnemo has a built-in system for extensions and games, which you can use to make your study experience a bit more fun.
Extentions and Games: The built-in system for extentions and games are fully open, meaning developers can create their own. If you are interested in creating your own, please refer to the plugins and extensions section.

4. Track Your Progress

Mnemo automatically tracks your study habits and progress, giving you a clear overview of your learning journey. View your stats directly on the dashboard through the statistics widgets.

NOTE: A seperate Analytics page is coming later.

The statistics collected involves:

  • Daily and weekly study time
  • Most productive hours
  • Subject-wise progress
  • Goal achievement rates
Privacy Notice: Mnemo does not collect any personal data, all data is stored locally on your device, except for information parsed to optional AI services.

Installation Guide

Last updated: July 17, 2025 4 min read

Follow these step-by-step instructions to install mnemo on your device. The process is straightforward and should take only a few minutes.

System Requirements

OS: Windows 10 (64-bit) or Windows 11
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: .NET Framework 4.7.2 or later
OS: macOS 10.15 (Catalina) or later
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: Apple Silicon and Intel Macs supported
OS: Ubuntu 18.04+, Debian 10+, or equivalent
RAM: 4GB minimum, 8GB recommended
Storage: 500MB free space
Display: 1280x720 minimum resolution
Network: Internet connection required for sync
Other: X11 or Wayland display server

Download Options

Choose the appropriate version for your operating system from our download page.

Installation Steps

  1. Double-click the downloaded mnemo_setup.exe file
  2. If prompted by Windows Defender, click "More info" and "Run anyway"
  3. Follow the installation wizard prompts
  4. Choose your installation directory (default recommended)
  5. Click "Install" and wait for completion
  1. Open the downloaded .dmg file
  2. Drag the mnemo icon to the Applications folder
  3. Right-click mnemo in Applications and select "Open"
  4. Click "Open" when prompted about unverified developer
  5. Grant necessary permissions when prompted

Using .deb (Ubuntu/Debian):

sudo dpkg -i mnemo.deb
sudo apt-get install -f

Using .rpm (Fedora/RHEL):

sudo rpm -i mnemo.rpm

Using .AppImage:

  1. Make the AppImage executable: chmod +x mnemo.AppImage
  2. Double-click to run or execute from terminal

Post-Installation

After installation, you should:

  1. Launch mnemo from your start menu or applications folder
  2. Take the tour to get familiar with the key features

Common Installation Issues

  • Windows Defender blocking: Use "Run anyway" option
  • macOS security warning: Open from Applications folder
  • Linux dependencies: Run sudo apt-get install -f

For more help, visit our troubleshooting guide.

First Study Session

Last updated: July 17, 2025 5 min read

Ready to start your first study session with mnemo? This guide will walk you through setting up and conducting your first productive study session, making the most of mnemo's powerful features.

Choosing Which Tool To Use

There are quite a lot of tools to choose from, here is a list:

  • Learning Paths - Create learning paths to help you study for exams or other important events. Learn more
  • Flashcards - Create flashcards to help you memorize important information. Learn more
  • Notes - Create notes to help you organize your study materials. Learn more
  • Quizzes - Create quizzes to help you test your knowledge. Learn more
  • Mindmaps - Create mindmaps to help you visualize and remember information. Learn more
Pro Tip: Start with short, focused sessions (25-30 minutes) to build momentum and avoid burnout.

Using a Study Timer

A lot of people use a study timer to stay focused and on track. We recommend adding the Timer extension from the extensions tab in the sidebar.

How to use the timer?

  1. Select a Timer Mode
    • Pomodoro (25 min work / 5 min break)
    • Custom timer (set your own time)
    • Flexible session (set your own time)
  2. Start Your Session
    • Click the play button to begin
    • Timer will track your focus time
    • Breaks are automatically scheduled
Break Time: Breaks are an important part of learning. Use them wisely! Stand up, stretch, or take a short walk. Avoid screens during breaks to reduce eye strain.

Review and Track Progress

After your study session:

  • Check your session statistics
  • Review completed tasks and goals
  • Schedule your next session
  • Set reminders for review sessions
Progress Tracking: Regular reviews of your study analytics help identify your most productive times and effective study methods.

Next Steps

Interface Overview

Last updated: July 17, 2025 5 min read

The interface is made to be simple and easy to use with more advanced features hidden around the corner. Lets take a look.

Sidebar

The sidebar contains all the main navigation options within the application. It is collapsable and expandable by clicking the hamburger menu icon in the top-left corner.

Expanded Sidebar

Collapsed Sidebar

Dashboard

The dashboard is the opening page of the application. It shows you your statistics and the option to jump back into a study session.

The dashboard is split into two sections;

  • The top section shows your statistics through widgets. You can add or remove widgets by clicking the icon in the top-right corner on the widget section then
  • The bottom section allows you to jump back into a session by double clicking on the session you want to jump back into. If you right click you get more options such as Go To, Export and Delete.

Dashboard

Learning Paths

Learning Paths is a great way to study for exams or tests. You can upload any reference materials you want to study from; including PDFs, text files, youtube videos, and more, or simply write a prompt and let the AI generate a learning path for you.

When the learning path is created, the first thing you will see is the unit overview. Most of them is grayed out, this is because the units are generated as you go. However, if you want to generate them all at once you can press the Generate All button, this allows you to also use the paths fully offline.

How To Use The Learning Paths

Each unit is divided into two sections: theory and practice. The theory section provides an article explaining the unit's concepts, while the practice options offer various ways to apply that knowledge.

A short overview of the practice options:

  • Quiz - Creates a quiz to test your knowledge based on the unit theory (can be transfered to the quizzes tab)
  • Flashcards - Creates flashcards to help you memorize the unit theory (can be transfered to the flashcards tab)
  • Adaptive - The adaptive tab is a bit different. It is meant to be a more complete and interactive way to learn. It splits up the unity theory into even smaller chunks, then give you a few questions based on what you just read, then repeat the process until you have learned the entireunit.

Games

Many studies have shown that games can be a great way to learn. Mnemo has a built-in game system where developers can create games and publish them. These games uses data from the mnemo application through the mnemo api.

Since games can pull from different data sources, not all games will be compatible with your preferred content. For instance, one game might use a flashcard deck, while another could rely on your mind maps.
Pro Tip: If you want to create your own game jump to the Games section.

How To Use The Games

In our games store, you can easily browse and discover games published to the Mnemo platform. Every game is free to download, and most don't even require an internet connection to play.

To install a game, just click the Install button. If you'd like more information before downloading, simply click the View button.

Got your own games? No problem! You can also import games you've created or downloaded from elsewhere. Just hit the Import button in the top-right corner of the store.

Games Store

Extensions

The application is built to be extensible. We did not want a application that was really busy and packed with unwanted features and bloat-ware. This is where the extensions system comes in to play.

Extentions is a way to add new features to the application that might not be needed for everyone. For instance, accessibility: not everyone wants or needs text to speech, accessibility fonts etc.

The extensions page is almost the same as the games store. You can browse for extentions published to the mnemo store and connect to them in just a single click. You do this by pressing the Connect button, and you can view more information about the extention by clicking the View button.

Pro Tip: If you want to create your own extension jump to the Extensions section.

Extensions Store

Settings

The settings page is where you are able to change things about the application. For instance, you can change the theme, language, and all the other settings you might want to change.

The settings menu is seperated into 5 different sections which you can navigate through with the tabs and the secondary sidebar. These sections are:

  • Apperance - Change the theme, font, and other apperance settings.
  • Preferences - Change the preferences for the application.
  • Data & Storage - Manage your data and storage.
  • About & Support - Information about the application and support.
  • Experimental - Experimental features that are not ready for production.
